# Product Overview

Yuxi-Know (语析) is an intelligent knowledge base and knowledge graph agent development platform powered by large language models.

## Core Capabilities

- **RAG Knowledge Base**: Document upload, processing, and intelligent retrieval using vector databases (Milvus)
- **Knowledge Graph**: Graph visualization and querying with Neo4j integration
- **Agent Framework**: Multi-agent system built on LangGraph v1 with middleware and sub-agent support
- **Document Processing**: OCR and parsing support via MinerU, PaddleX, and RapidOCR
- **Evaluation System**: RAG evaluation with benchmark support (Milvus knowledge bases only)
- **Multi-modal Support**: Image and text processing capabilities

## Key Features

- Light/dark mode UI
- Dashboard for usage analytics
- Knowledge graph visualization (LightRAG support)
- Mind map generation from knowledge base files
- File/folder/zip upload support
- Custom model provider configuration
- MCP (Model Context Protocol) server integration

## Target Users

Developers building intelligent agent platforms with RAG and knowledge graph capabilities. MIT licensed for customization and extension.
